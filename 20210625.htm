1. ೧೪ ಮನ್ವಂತರಗಳ ವಿವರ

ಕ್ರ.ಸ. ಮನು ಮಕ್ಕಳು ಇಂದ್ರ ದೇವಗಣ ಋಷಿಗಣ ಅವತಾರ
೧ ಸ್ವಾಯಂಭುವ ಉತ್ತಾನಪಾದ, ಪ್ರಿಯವ್ರತ ಯಜ್ಞದೇವ ತುಷಿತ.. ಮರೀಚಿ, ಅತ್ರಿ, ಅಂಗೀರಸ, ಪುಲಸ್ತ್ಯ, ಪುಲಹ, ಕ್ರತು, ಭೃಗು (ರುಚಿ + ಅಕುತಿ = ಯಜ್ಞ), (ಕರ್ದಮ + ದೇವಾಹುತಿ = ಕಪಿಲ)
೨ ಅಗ್ನಿ-> ಸ್ವಾರೋಚಿಷ ದ್ಯುಮಾನ್, ಸುಷೇಣ, ರೋಚಿಷ್ಮಾನ್ ರೋಚನ ತುಷಿತ.. ಊರ್ಜಸ್ತ೦ಬ.. ವೇದಶಿರಾ + ತುಷಿತೆ = ವಿಭು _ ನೈಷ್ಠಿಕ ಬ್ರಹ್ಮಚಾರಿ
೩ ಪ್ರಿಯವ್ರತ-> ಉತ್ತಮ ಪವನ, ಸೃ೦ಜಯ, ಯಜ್ಞಹೋತ್ರ.. ಸತ್ಯಜಿತ್ ಸತ್ಯ, ವೇದಶ್ರುತ, ಭದ್ರ, ಸತ್ಯವ್ರತ ವಸಿಷ್ಠ-> ಪ್ರಮದ.. (೭) ಧರ್ಮ + ಸೂನೃತ = ಸತ್ಯಸೇನ
೪ ಪ್ರಿಯವ್ರತ-> ತಾಮಸ ಪೃಥು, ಖ್ಯಾತಿ, ನರ, ಕೇತು.. (೧೦) ತ್ರಿಶಿಖ ಸತ್ಯಕ, ಹರಿ, ವೀರ +ವಿದೃತಿ-> ವೈದೃತಿ ಜ್ಯೋತಿರ್ಧಾಮ ಹರಿಮೇಧಾ + ಹರಿಣಿ = ಹರಿ
೫ ಪ್ರಿಯವ್ರತ-> ರೈವತ ಅರ್ಜುನ, ಬಲ, ವಿಂದ್ಯ ವಿಭುು ಭೂತರಯ.. ಹಿರಣ್ಯರೋಮಾ, ವೇದಶಿರಾ, ಊರ್ಧ್ವೇಭಾಹು .. (೭) ಶುಭ್ರ + ವಿಕುಂಠಾ = ವೈಕುಂಠ
೬ ಚಕ್ಷು->ಚಾಕ್ಷುಷ ಪೂರು, ಪುರುಷ, ಸುದ್ಯುಮ್ನ.. ಮಂತ್ರದ್ರುಮ ಆಪ್ಯ.. ಹವಿಷ್ಮಾನ್, ವೀರಕ.. ವೈರಾಜ + ಸಂಭೂತಿ = ಅಜಿತ
೭ ವಿವಸ್ವಂತ-> ಶ್ರಾದ್ಧದೇವ (ವೈವಸ್ವತ) ಇಕ್ಷ್ವಾಕು, ನಭಗ, ದೃಷ್ಟ, ಶರ್ಯಾತಿ, ನರಿಷ್ಯಂತ, ನಾಭಾಗ, ದಿಷ್ಟ, ಕರೂಷ, ಪೃಷಧ್ರ, ವಸುಮಾನ ಪುರಂದರ ಆದಿತ್ಯ, ವಸು, ರುದ್ರ, ವಿಶ್ವೇದೇವ, ಮರುದ್ಗಣ, ಅಶ್ವಿನಿಕುಮಾರರು, ಋಭು ಕಶ್ಯಪ, ಅತ್ರಿ, ವಸಿಷ್ಠ, ವಿಶ್ವಾಮಿತ್ರ, ಗೌತಮ, ಜಮದಗ್ನಿ, ಭಾರದ್ವಾಜ ಕಶ್ಯಪ + ಅದಿತಿ = ವಾಮನ
೮ ವಿವಸ್ವಂತ-> ಸಾವರ್ಣಿ ನಿರ್ಮೋಕ, ವಿರಜಸ್ಕ.. ವಿರೋಚನ-> ಬಲಿ ಸುತಪಾ, ವಿರಜಾ, ಅಮೃತಪ್ರಭ ಗಾಲವ, ದೀಪ್ತಿಮಾನ್, ಪರಶುರಾಮ, ಅಶ್ವತ್ಥಾಮ, ಕೃಪಾಚಾರ್ಯರು, ಋಷ್ಯಶೃಂಗ, ವೇದವ್ಯಾಸ ದೇವಗುಹ್ಯ + ಸರಸ್ವತಿ = ಸಾರ್ವಬೌಮ
೯ ವರುಣ-> ದಕ್ಷಸಾವರ್ಣಿ ಭೂತಕೇತು, ದೀಪ್ತಕೇತು .. ಅದ್ಭುತ ಪಾರ, ಮರೀಚಿಗರ್ಭ ದ್ಯುತಿಮಾನ್.. ಆಯುಷ್ಮ೦ತ + ಅಂಬುಧಾರ = ಋಷಭ _ ಕಲಾವತಾರ
೧೦ ಉಪಶ್ಲೋಕ->ಬ್ರಹ್ಮಸಾವರ್ಣಿ ಭೂರಿಷೇಣ.. ಶಂಭು ಸುವಾಸನ, ವಿರುದ್ಧ ಹವಿಷ್ಮಾನ್, ಸುಕೃತಿ, ಸತ್ಯ, ಜಯ, ಮೂರ್ತಿ.. ವಿಶ್ವಸೃಜ + ವಿಷೂಚಿ = ವಿಶ್ವಕ್ ಸೇನಾ
೧೧ ಧರ್ಮಸಾವರ್ಣಿ ಸತ್ಯ, ಧರ್ಮ.. (೧೦) ವೈದೃತ ವಿಹಂಗಮ, ಕಾಮಗಾಮ, ನಿರ್ವಾಣರುಚಿ ಅರುಣ.. ಆರ್ಯಕ್ + ವೈದೃತಿ = ಧರ್ಮಸೇತು
೧೨ ರುದ್ರಸಾವರ್ಣಿ ದೇವವಾನ್, ಉಪದೇವ, ದೇವಶ್ರೇಷ್ಠ ಋತಧಾಮ ಹರಿತ .. ತಪೋಮೂರ್ತಿ, ತಪಸ್ವೀ, ಅಗ್ನೀಧ್ರಕ.. ಸತ್ಯಸಹಾಯ + ಸೂನೃತ = ಸ್ವಧಾಮ
೧೩ ದೇವಸಾವರ್ಣಿ ಚಿತ್ರಸೇನ, ವಿಚಿತ್ರ ದಿವಸ್ಪತಿ ಸುಕರ್ಮ, ಸುತ್ರಾಮ ನಿರ್ಮೊಕ, ತತ್ವದರ್ಶ.. ದೇವಹೊತ್ರ + ಬೃಹತೀ = ಯೋಗೇಶ್ವರ
೧೪ ಇಂದ್ರಸಾವರ್ಣಿ ಉರು, ಗಂಭೀರಬುದ್ದಿ ಶುಚಿ ಪ್ರವಿತ್ರ, ಚಾಕ್ಷುಷ ಅಗ್ನಿ, ಬಾಹು, ಶುಚಿ, ಶುದ್ಧ, ಮಾಗಧ.. ಸತ್ರಾಯಣ + ವಿತಾನಾ = ಬೃಹದ್ಭಾನು
